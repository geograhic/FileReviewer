# 给朋友的快速上手

## 1. 打开

双击：

`dist\智能文件复习系统2.12.0_WebUI.exe`

它会打开一个桌面软件窗口，不会弹黑色命令行窗口。首次启动可能需要等待几秒。

## 2. 第一次使用

1. 跟随欢迎向导。
2. 点击“选择文件库”，选择一个已有资料文件夹。
3. 如果没有弹出选择窗口，就在“文件库”页左侧把文件夹路径粘贴进去，再点“扫描路径”。
4. 扫描完成后，在“文件库”里浏览和搜索资料。
5. 每天点击“开始复习”，读完后选择“忘记 / 困难 / 良好 / 简单”。

软件不会移动或删除你的原始文件。删除索引记录也不会删除原文件。

复习开始时，软件默认只在内部预览，不会自动弹出本地默认软件。遇到不能预览的格式，可以点“默认打开”或“其他应用”。

## 3. 写笔记

- 在“笔记”页可以直接新建和保存笔记。
- “本地方式新建”会创建真实 Markdown 文件，并打开所在文件夹。
- 笔记默认放在主数据目录的 `notes\` 文件夹，也可以在“设置”里自定义路径。
- 笔记列表左侧可以勾选，多选后可以“导出所选”或“删除所选”。
- 删除笔记会删除 Markdown 文件和数据库记录，软件会先让你确认。

## 4. 切换语言

进入“设置”，在“语言”里选择：

- 简体中文
- English

## 5. 数据在哪里

默认主数据目录：

`C:\Users\你的用户名\Documents\LiFileReviewer2`

设置页会显示配置、数据库、备份、日志、插件目录、笔记目录的完整路径。你也可以在设置页把主数据目录或笔记目录迁移到自己指定的位置。主数据目录可以迁移到空文件夹；如果选择的是已有普通文件夹，软件会自动创建 `LiFileReviewer2` 子目录，不会打乱那个文件夹里的原有文件。

导出目录也可以自定义。2.9.0 起，CSV、JSON、迁移包、数据库备份、分享包、笔记导出和导入前备份都会先弹出选择窗口，让你当场决定保存到哪里；默认导出目录只是让选择窗口先打开到那个位置。

如果你的文件库路径很深、文件名很长，软件会自动使用 Windows 长路径兼容。创建关联笔记时，标题会尽量保留完整；必要时只缩短真实文件名。

## 6. 换电脑或备份

在“设置”页点击：

- “选择导出目录”：设置保存窗口默认打开的位置
- “备份数据库”：选择一个 `.sqlite` 位置后备份复习数据库
- “导出 JSON”：选择一个 `.json` 位置后导出开放数据
- “导出迁移包”：选择一个 `.zip` 位置后导出完整配置、数据库备份、插件目录、笔记目录
- “导入迁移包”：在另一台电脑导入

推荐每隔一段时间点一次“体检”和“导出迁移包”。

## 7. 成就与插件

- 成就不再只是固定清单，会显示 XP、等级、称号和进度。
- 成就系统现在是“成就系统”插件，可以在设置页插件管理里启用或关闭。
- 关闭成就插件后，总览页不会再保留成就占位；重新启用后会自动显示。
- 内置成就会随着复习次数、笔记、完成资料持续升级。
- 以后可以把只含 `plugin.json` 的成就包放进 `plugins\`，追加更多里程碑。
- 插件管理里可以“导入插件包”“导入文件夹”，也可以一键打开插件目录查看文件。
- 默认只读取插件 JSON，不执行插件代码，稳定优先。

## 8. 社交资料插件

- 设置页插件管理里可以启用或关闭“社交资料”插件。
- 启用后可以填写显示名称、账号 ID、简介、主页、联系信息和分享偏好。
- 关闭社交资料插件后，设置页不会再显示社交资料表单；重新启用后会自动显示。
- “复制社交名片 JSON”会生成本地名片数据，为未来好友、动态、协作学习功能做准备。
- 当前版本不会自动上传社交资料。
