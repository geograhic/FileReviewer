# 智能文件复习系统 2.0 WebUI

这是独立新版本，不会修改 1.x 原项目文件夹。当前应用版本为 `2.5.0`，默认以桌面窗口运行，不再像普通网页一样弹浏览器，也不会显示黑色命令行窗口。

## 启动

- 打包版：双击 `dist\智能文件复习系统2.0_WebUI.exe`
- 源码版：双击 `启动_智能文件复习系统2.0.bat`
- 调试浏览器模式：`python app.py --browser`
- 纯本地服务模式：`python app.py --no-window --port 8897`

应用内部只监听本机 `127.0.0.1`。如果默认端口 `8765` 被占用，会自动尝试后续端口。

## 数据与配置

默认主数据目录：

`C:\Users\你的用户名\Documents\LiFileReviewer2`

所有关键数据集中放在这个目录里：

- `config.json`：用户配置
- `review_data.sqlite`：复习数据库
- `backups\`：数据库备份
- `plugins\`：未来插件目录
- `notes\`：默认笔记目录，保存真实 Markdown 文件
- `exports\`：默认导出目录，保存 CSV、JSON、迁移包、手动备份和笔记导出
- `app.log`：运行日志
- `runtime.json`：本次启动的本地服务信息

设置页支持直接自定义“主数据目录”“笔记目录”和“默认导出目录”，也支持一键导出迁移包 `LiFileReviewer2_profile_*.zip`。迁移后，默认目录里只保留一个很小的 `profile_location.json`，用于下次启动时找到新的主数据目录。

## 主要能力

- 直接选择本地文件夹作为文件库，像 Obsidian 一样浏览已有资料
- 如果原生文件夹选择器不可用，可直接粘贴文件夹路径并扫描
- 批量扫描 PDF、Word、PPT、Excel、Markdown、HTML、图片、音视频等资料
- 批量设置标签、暂停、恢复、完成、设为今天复习、删除索引
- 复习算法支持 FSRS-Lite、SM-2、固定间隔
- 四档评价：忘记、困难、良好、简单
- 复习开始默认不再自动调用系统默认程序，避免干扰阅读
- 无法内嵌预览的文件可手动选择“默认打开”或“其他应用”
- 支持软件内新建/编辑笔记，也支持按本地文件方式创建 Markdown 笔记
- 笔记存储路径可自定义，支持勾选、全选、取消选择、批量导出和确认删除
- 笔记随迁移包一起备份迁移；导入迁移包后会自动修正笔记路径到当前电脑的数据目录
- 自动记录阅读时长、复习次数、历史和下次到期时间
- 支持提醒、主题、强调色、自定义 CSS、中英界面
- 设置页支持自定义导出目录、体检、数据库备份、CSV/JSON 导出、完整迁移包导出/导入
- 独立“帮助”页内置文件库、复习、笔记管理、导出迁移、路径配置、插件和维护说明
- 预留 `plugins\` 插件目录和插件清单读取能力，后续可持续扩展

## 安全说明

扫描文件库只读取路径和文件元数据，不移动、不复制原始资料。删除索引记录也不会删除原始文件。删除笔记会删除用户创建的 Markdown 笔记文件和数据库记录，软件会先弹出确认。

## 打包

```powershell
python -m PyInstaller --noconfirm --clean --onefile --windowed --name "智能文件复习系统2.0_WebUI" --add-data "web;web" --hidden-import webview.platforms.winforms app.py
```

打包完成后运行：

```powershell
dist\智能文件复习系统2.0_WebUI.exe
```
